- 所有 Ruby 插件统一使用 `Jekyll::Hooks.register <scope>, :<phase> do |post/site| ... end` 注册钩子，避免自定义 Liquid 标签。
- HTML 输出中所有用户可控文本一律经 `| escape` 过滤，URL 一律经 `| relative_url` / `| absolute_url` 处理。
- 可选功能通过 `{%- if site.xxx -%}` 或 `{%- if page.comments != false -%}` 包裹，保持主题开箱即用且无外部依赖时仍可编译。
- 页面内交互逻辑以 IIFE 形式内联在模板末尾，通过 `querySelector` 选择 DOM 节点后先判空再绑定事件。
