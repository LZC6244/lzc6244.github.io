本模块为纯静态资源目录，按用途划分为三个并列子目录：
- `assets/css/search.css` 与 `assets/js/search.js` 实现站内全文搜索的弹窗界面与客户端索引匹配逻辑（search.js 通过 `data-search-url` 拉取 JSON 索引，支持英文单词边界 + 中文二元组模糊匹配、分页懒加载与滚动加载更多）。
- `assets/img/avatar.jpg`、`qrcode.jpg` 为页面引用的头像与二维码图片。
- `favicons/` 由 https://realfavicongenerator.net/ 生成，包含各平台图标、`site.webmanifest` 与 `browserconfig.xml`，配合 `_includes/head.html` 中的 meta 标签生效；`README.md` 记录了替换流程。
- `files/` 按年份子目录归档可下载的配置文件与脚本（如 Milvus compose、PostgreSQL 配置、iptables 白名单脚本），命名采用 `YYYY-MM-DD-NNN-标题.扩展名` 的有序前缀，供文章内 `<a href="/files/...">` 引用下载。
