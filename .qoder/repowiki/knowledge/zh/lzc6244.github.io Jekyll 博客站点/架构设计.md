根目录作为 Jekyll 站点的装配层：_config.yml 声明 theme: minima、permalink 格式（/:year/:month/:day/:title.html）、markdown: kramdown 与 highlighter: rouge，并启用 jekyll-sitemap/jekyll-seo-tag/jekyll-feed 三个官方插件；Gemfile 在本地 Ruby ≥ 4.0 时绕过 github-pages gem 直接锁定 jekyll ~3.9 + minima ~2.5 + liquid >= 4.0.4，以兼容 GitHub Actions 的 Ruby 3.3.4 环境。运行时各子模块按约定被 Jekyll 发现——posts/ 提供 Markdown 源，theme_assets/ 覆盖主题布局与片段，static_assets/ 注入 CSS/JS/图片，site_pages/ 生成 RSS 与搜索索引等静态页面，最终由 GitHub Pages 流水线产出 https://lzc6244.github.io/。
