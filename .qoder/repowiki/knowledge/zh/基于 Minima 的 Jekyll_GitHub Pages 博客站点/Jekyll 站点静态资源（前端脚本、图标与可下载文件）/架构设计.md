本模块为纯静态资源目录，无构建逻辑，由 Jekyll 原样复制到 `_site`：
- `assets/css/search.css` 与 `assets/js/search.js` 实现站内全文搜索弹窗：search.js 通过 `<input data-search-url>` 拉取 JSON 索引，在客户端对英文关键词做单词边界匹配、对中文连续字符做二元组模糊评分，并支持分页懒加载与滚动加载更多；CSS 仅负责弹窗遮罩、面板布局与高亮样式。
- `assets/img/avatar.jpg`、`qrcode.jpg` 为页面直接引用的头像与二维码图片。
- `favicons/` 是 realfavicongenerator.net 的一次性生成产物，包含各尺寸 PNG/SVG、`favicon.ico`、`browserconfig.xml` 与 `site.webmanifest`，配合 `_includes/head.html` 中的 meta 标签生效；`README.md` 记录了替换流程。
- `files/` 按年份子目录归档可下载的配置文件与脚本（如 Milvus compose、PostgreSQL 配置、iptables 白名单），命名采用 `YYYY-MM-DD-NNN-标题.扩展名` 的有序前缀，供文章内 `<a href="/files/...">` 引用下载。
