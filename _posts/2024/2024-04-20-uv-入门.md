# 换源

## 更换清华源

在项目 pyproject.toml 文件最下方添加

```TOML
[[tool.uv.index]]
# 更换为阿里源
url = "https://mirrors.aliyun.com/pypi/simple"
default = true
```





# 安装 python 虚拟环境和依赖

```bash
# 使用指定 python 创建虚拟环境和安装依赖
uv sync --python 3.12.12



# 使用当前 python 创建虚拟环境和安装依赖
uv sync --python $(which python)
```
