# 常用命令



```bash
less -smruI xxx.log
```



- **\-s 或 \-\-squeeze\-blank\-lines:** 将多个连续的空行压缩成一行显示，使文件更紧凑。

- **\-m 或 \-\-long\-prompt:** 显示更详细的提示信息，包括当前行号、总行数和文件百分比。

- **\-r 或 \-\-raw\-control\-chars:** 将控制字符以原始形式显示，例如 ^M 符号。

- **\-u 或 \-\-unicode:** 将文件内容解释为 UTF\-8 编码，并正确显示特殊字符。

- **\-I:** 搜索忽略大小写



# 切换文件

less 查看多个文件时怎么切换文件

- `:n` ：切换到下一个文件

- `:p` ：切换到上一个文件





# 处理中文乱码

配置环境变量

```bash
export LC_ALL=en_US.UTF-8
export LESSCHARSET=utf-8
```
