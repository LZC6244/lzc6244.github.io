# 常见用法

```Shell
# 新建指定名称 screen 会话
screen -S xxxx会话名


# 查看已有 screen 会话
screen -ls


# 恢复指定名称 screen 会话
screen -r xxxx会话名


# 关闭指定名称 screen 会话
# 进入指定名称 screen 会话后，通过 Ctrl+D 快捷键退出关闭会话
```





# 在 screen 会话中使用鼠标进行上下滚动



1. 进入回滚模式，在 screen 会话中按下快捷键：<kbd>Ctrl+A</kbd>

2. 此时就已经可以通过鼠标滚轮进行上下滑动了

3. 退出回滚模式，按下 <kbd>Esc</kbd> 即可
