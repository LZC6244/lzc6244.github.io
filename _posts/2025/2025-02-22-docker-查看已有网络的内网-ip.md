# shell 命令

查看已有网络的内网 ip 并根据 ip 进行升序排序

```bash
docker network inspect $(docker network ls -q) --format "{{.Name}}{{range .IPAM.Config}} : {{.Subnet}}{{end}}" | sort -t ":" -k "2,2" -V | less -smruI
```



## 命令解释

根据 **\|** 管道进行命令解释



- <code class="cmd">docker network inspect $(docker network ls -q) --format "{{.Name}}{{range .IPAM.Config}} : {{.Subnet}}{{end}}"</code>

  - `docker network ls -q`：列出所有 Docker 网络的 ID。

  - `docker network inspect`：根据网络 ID 详细检查网络信息。

  - `--format '{{.Name}}{{range .IPAM.Config}}:{{.Subnet}}{{end}}'`：自定义输出格式，使用冒号 `:` 分隔子网名称和子网 IP，将子网名称和子网 IP 组合成 `子网名称:子网 IP` 的形式输出。





- <code class="cmd">sort -t: -k2,2V</code>

  - `sort`：用于对文本进行排序。

  - `-t ":"`：指定分隔符为冒号 `:`，这样 `sort` 命令会根据指定的分隔符来划分字段。

  - `-k "2,2V"`：指定排序的字段为第 2 个字段（即子网 IP 部分），`2,2` 表示只对第 2 个字段进行排序。

  - `-V`：按照版本号的规则进行排序，适合对 IP 地址和子网进行排序，能正确处理 IP 地址的数字顺序。





- <code class="cmd">less -smruI</code>

  - 通过 `less` 命令来查看输出
