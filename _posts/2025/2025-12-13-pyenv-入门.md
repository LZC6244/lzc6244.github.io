# 概要

pyenv git 地址：https://github.com/pyenv/pyenv



# 安装 pyenv

```bash
# 安装 pyenv
curl https://pyenv.run | bash
 
# 在 ~/.bashrc 末尾添加以下 pyenv 配置，配置其自动初始化和环境变量（poetry 安装后会提示）
export PYENV_ROOT="$HOME/.pyenv" 
[[ -d $PYENV_ROOT/bin ]] && export PATH="$PYENV_ROOT/bin:$PATH" 
eval "$(pyenv init -)" 
eval "$(pyenv virtualenv-init -)"
 
# 应用 ~/.bashrc 配置
source ~/.bashrc
 
 
# （建议提前换源）提前安装系统依赖（ubuntu）
sudo apt update
sudo apt install make build-essential libssl-dev zlib1g-dev  libbz2-dev libreadline-dev libsqlite3-dev curl git  libncursesw5-dev xz-utils tk-dev libxml2-dev libxmlsec1-dev libffi-dev liblzma-dev -y
 
 
# 编译安装 python3.10 ，每次运行会从 python 官网下载安装包，无缓存
# 由于编译安装可能会出现缺失系统依赖情况，请手动下载相应 python 安装包
# 创建缓存文件夹 mkdir -p ~/.pyenv/cache 后将安装包压缩包存入此路径即可
pyenv install 3.10

# 全局配置 python 版本为 3.10
pyenv global 3.10
# 在当前路径配置 python 版本为 3.10
# pyenv local 3.10

# 查看、确认当前 pyenv 配置的 python 版本
pyenv versions


# 创建虚拟环境
pyenv virtualenv 3.12 my-venv
```





# 更新 python 版本列表

寻找不知道指定/最新 python 版本时执行更新 python 版本列表操作

```bash
pyenv update
```





# 安装 python



## 直接联网安装

```bash
# 安装指定版本 python
pyenv install 3.10.11
# 列出 pyenv 当前已安装可用 python 版本
pyenv versions
```



## 通过本地 python 安装包缓存安装

python-build 会将下载的包文件缓存在`PYTHON_BUILD_CACHE_PATH`环境变量指定的位置。

默认值为`~/.pyenv/cache`，因此只需创建该目录即可启用缓存。



### linux

创建 `~/.pyenv/cache` 然后下载指定 python 版本的 tar.xz 源码压缩包即可



### windows

Windows 使用的是 pyenv-win ，创建 `C:Users用户名.pyenvpyenv-wininstall_cache` 后下载指定 python 版本的 exe 即可





### 步骤

以 windows 为例，如下

```Dockerfile
# 安装指定版本 python
pyenv install 3.10.11
# 列出 pyenv 当前已安装可用 python 版本
pyenv versions

# 配置全局或者当前路径 python 版本
# 指定全局 python 版本
pyenv global 3.10.11
# 指定当前路径 python 版本
pyenv local 3.10.11

# 校验全局 python 版本配置
python -c "import sys;packages_path='site-packages';print(f'Python 版本：{sys.version}nPython 路径：{sys.executable}nsite-packages 路径：{next(path for path in sys.path if packages_path in path)}')"
```
