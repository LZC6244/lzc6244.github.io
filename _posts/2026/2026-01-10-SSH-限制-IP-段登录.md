# 步骤

- 编辑 ssh 配置文件：`sudo vim /etc/ssh/sshd_config`

加入下面内容

```bash
# 多条记录表示多条规则
# * 表示所有用户
AllowUsers *@xx.xx.xx.0/24
# 允许 user 用户从 xx.xx.xx.0/24 网段访问
AllowUsers user@xx.xx.xx.0/24
```

- 验证配置是否正确：`sudo sshd -t`



- 重启 ssh 服务

<span style="color:red">重启前建议保留当前 ssh 连接，避免 ip 段配置错误</span>

<span style="color:red">ssh 连不上就完了，这样还能用已有 ssh 连接进行挽救</span>

执行命令：`sudo systemctl restart ssh`
